# BEM

Block, Element, Modifer 의 약자로, 클래스를 명명하는 방법/규칙

```
Block__Element--modifer
```

## Block

독립적으로 의미를 갖는 객체로, 쉽게 Component 라고 생각하자

## Element

독립적으로 의미를 갖지못하는, Block 을 구성하는 요소

## Modifier

Block이나 Element의 flag로 동작이나 모습에 대한 속성 값을 나타낸다.
