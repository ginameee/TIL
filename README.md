# [Today I Learn](https://ginameee.github.io/TIL)

하루하루 배우고 느낀걸 기록하는 공간입니다. <br>
[docsify.js](https://docsify.js.org/#/)를 사용해 만들어졌습니다.

## 로컬 작업 시

### 설치

```
> npm i -g docsify
```

### 실행

```
> cd {clone경로}
> docsify serve
```

## 규칙

- TIL이지만 날짜별로 정리하진 않습니다.
- 주제별로 정리합니다.
- 파일 네이밍은 **[YYYYMMDD]\_[글제목]** 으로 합니다.
